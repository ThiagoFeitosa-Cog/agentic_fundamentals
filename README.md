# Recruitment Assistant

This repository contains the Define phase artifacts for a Recruitment Assistant built using the AAMAD methodology.

## Project Overview

The Recruitment Assistant is a multi-agent application designed to help recruiters source, evaluate, and recommend candidates based on job requirements.

## Problem Statement

Manual candidate sourcing and evaluation can be slow, inconsistent, and difficult to scale. This project explores how a multi-agent AI system can support recruiters by coordinating candidate research, evaluation, and recommendation workflows.

## Key Features

- Candidate sourcing based on job requirements
- Candidate evaluation against role criteria
- Ranked candidate recommendations
- Explanation of candidate strengths, gaps, and fit

## Application Crew

The Application Crew includes:

- Researcher Agent: Finds potential candidates.
- Evaluator Agent: Evaluates candidates against job criteria.
- Recommender Agent: Produces ranked recommendations.
