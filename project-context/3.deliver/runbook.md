# Operational Runbook
## Recruitment Assistant

## Application Overview

The Recruitment Assistant is a lightweight multi-agent recruitment workflow application. It evaluates candidate profiles against role requirements and returns ranked recommendations.

## Prerequisites

- Python 3.8+
- pip
- Browser
- Git
